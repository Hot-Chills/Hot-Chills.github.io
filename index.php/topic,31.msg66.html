I have been doing some small script updates during the last few days:
  • Foul language is now censored and the player is slapped. Unfortunately, this has caused spamming among dead and spectating players, who can't get slapped. I might have to do something about that.
  • CS2D 0.1.2.2 made it possible to use multiple colours in text. I have used that to make the chat lines more readable. Tags are drawn in a separate colour, for instance.
  • A few new rounds were added and a few modified. DC changed the behaviour of barricades, which made a few tweaks necessary. It also opened up new possibilities.

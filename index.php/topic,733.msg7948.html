No, DoS attacks are simply overloading the server with a request so many times that it can't handle the load, which results in the server to temporarily shut down.

In proper English:
Quote
saturating the target machine with external communications requests, so much so that it cannot respond to legitimate traffic, or responds so slowly as to be rendered essentially unavailable. Such attacks usually lead to a server overload.

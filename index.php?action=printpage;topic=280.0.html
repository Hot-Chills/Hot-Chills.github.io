Title: U.S.G.N. names
Post by: Häppy C@mper on September 28, 2013, 15:33:29
It's time for a new script update. This time I added a feature inspired by a script by Banaan: to fetch U.S.G.N. names directly from Unreal Software's web site. The names are displayed where it makes sense, mostly in administration menus, but also in the Identify command. Finally I replaced the "Online Users" command with a new one, "List Players", which lists all players online and also show their U.S.G.N. names. It's available for Junior Moderators and higher.

The script will be activated soon.

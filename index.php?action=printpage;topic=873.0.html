Title: Language rule
Post by: AREIOS on May 12, 2015, 11:18:29
Hello,

This suggestion about a possible new rule. It happens quite often that two or even more players speak the same language (apart from english) and they choose to do so in the main chat.

For someone who doesn't speak that language it looks like nonsense, right? And if it was nonsense, it would be considered spam, which is not allowed. Following that logic I would suggest to add a rule to the server info that says something like "Speak only english in the main/general chat".

There is a pm function to which (as far as I know) everyone have access.

In case it is already a rule, please make it visible because its obviously not clear for the average player and we can't refer to them as easilly as "Press F1".

Since our set of rules are written in english, there are players who wouldn't understand this new rule and therefor not knowing they are breaking it. For those there is not much to do other than warning, eventually muting and hoping he/she gets the point. In my opinion its more important that most players are not annoyed with full-screen nonsense than a few player's privilige to using their mother tongue.

Obviously one or two line from any language is not an issue, I'm talking about long conversations.

What do you think?

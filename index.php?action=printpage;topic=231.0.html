Title: Dynamic Server IP Address?
Post by: Nighthawk on August 24, 2013, 08:22:30
Hi,
I think this might pull down the % of ddos attacks on the server. And that the dynamic IP would change every 2-5 hours, or if the ddos attack is frequent, then every hour. This might be a problem for the players, but who cares? I still love this server, and it will/might protect it, because ddos needs time to send enough packets to crash a computer, not 1 hour, it might take days.

And the best way for me to make a dynamic IP address is:
ipconfig /flush
ipconfig /renew

Takes a few seconds, then you're good to go. It would be easy if you could make a batch file like this, then download a "cron jobs" type software to make it run, or I think there is already a system like that on windows. (I saw it once on XP, cant be sure it will be on higher versions).

Cheers,
Nighthawk

Title: Re: smth about scripting
Post by: Nighthawk on July 12, 2014, 09:25:05
For a website you will need to learn HTML5 and CSS3. On top of that for basic page functions as you stated to be made from scratch (which isn't hard at all) you will need to learn PHP. You can find many books and online tutorials on PHP and web designing. Start with HTML and CSS, when you're done learning designing move on to PHP for functions like that. And while you learn PHP make use of mySQL, which is a database container.

To make functions like you suggested PHP is required, to use and output the results of your PHP code you need a PHP/Mysql/Apache environment such as Wamp or XAMPP. Either way you can find tons of tutorials and books about getting started. Good luck.
